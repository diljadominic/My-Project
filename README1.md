Emotion Detection Through Social Media Posts

Description
This project analyzes user-generated social media posts using Natural Language Processing (NLTK) to detect emotional patterns such as sadness and depression. When repeated negative emotions or signs of depression are identified, the system enables the admin to take appropriate action by recommending professional psychological support.
The platform also provides users with the option to privately consult a psychologist if they feel the need for help. It is designed with a strong focus on privacy, allowing individuals to express their feelings without fear of judgment, while helping emotionally vulnerable users receive timely support through intelligent analysis.

Technologies Used
- Python
- Django
- NLTK (Natural Language Processing)
- Machine Learning

Features
- Detects emotions from user text posts in real-time  
- Allows users to request private consultation with a psychologist  
- Admin dashboard to monitor user emotions and take action  
- Secure and privacy-focused user interaction system  

How to Run
1. Clone the repository  
2. Install required packages using `pip install -r requirements.txt`  
3. Run migrations using `python manage.py migrate`  
4. Start the server using `python manage.py runserver`  
5. Open browser and go to `http://127.0.0.1:8000/`
